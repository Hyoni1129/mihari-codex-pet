# Mihari Codex Pet

A fan-made, unofficial Codex pet inspired by Mihari Oyama from ONIMAI.

This is a non-commercial fan project for personal customization of the Codex pet overlay.

## Update notes

- Improved `running-left` and `running-right` from the earlier short-feeling motion into smoother 8-frame directional movement.
- Made the Codex task states easier to distinguish: `running` now shows Mihari writing research notes in a small notebook, while `review` shows her checking output with a compact clipboard.
